## Proxy Pattern
>Intercept and control interactions to target objects

- ターゲットとなるオブジェクトを直接操作する代わりに、Proxy（代役）オブジェクトとやり取りする
- プロキシは、オブジェクトの振る舞いを制御するための手段としてよく使う
  - バリデーション、通知、デバッグ
- パフォーマンスが重要なコードにはプロキシを使用しない
